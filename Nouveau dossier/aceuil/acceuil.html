A PROPOS

Ce site a été créé par Benpro max pour que quiconque puisse apprendre la programmation web. Il y a trois principaux langages pour la programmation web qui sont : Le HTML qui est utilisé pour la création et la représentation le contenu d’une page web et sa structure Le CSS correspond à un langage informatique permettant de mettre en forme les pages web (HTML) Le JAVASCRIPT est utiliser pour une interaction avec : les formulaires, les boutons etc.
